**HyperCardHackerNews** is an experimental web application that simulates what
Hacker News might have looked like if it existed in the HyperCard era of the
early 1990s. Navigate through stories with authentic HyperCard-style buttons,
draw on cards with the pencil tool, and experience the nostalgic charm of
black-and-white bitmap interfaces.

Despite the retro aesthetic, this is a modern React application:

- **React 18** for component-based UI
- **TypeScript** for type safety
- **Vite** for blazing-fast builds
- **Tailwind CSS** for styling (yes, really!)
- **Web Audio API** for authentic system sounds
- **Pure frontend** - no backend required, all data is hardcoded

The entire HN dataset is stored in `constants.ts` for that authentic
"stack-based data storage" feel.
